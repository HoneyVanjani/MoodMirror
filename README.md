### `npm run eject`

**Note: this is a one-way operation. Once you `eject`, you can't go back!**

If you aren't satisfied with the build tool and configuration choices, you can `eject` at any time. This command will remove the single build dependency from your project.

Instead, it will copy all the configuration files and the transitive dependencies (webpack, Babel, ESLint, etc) right into your project so you have full control over them. All of the commands except `eject` will still work, but they will point to the copied scripts so you can tweak them. At this point you're on your own.

You don't have to ever use `eject`. The curated feature set is suitable for small and middle deployments, and you shouldn't feel obligated to use this feature. However we understand that this tool wouldn't be useful if you couldn't customize it when you are ready for it.

# MoodMirror V2.0 🧠✨
### *The Premium Neural Architecture for Emotional Intelligence*

MoodMirror V2.0 is a state-of-the-art emotional wellness ecosystem designed with a high-fidelity "Neural Architecture." It bridges the gap between biological sentiment and digital clarity, providing users with a glassmorphic, high-contrast interface to track, analyze, and synchronize their emotional well-being.

---

## 🌌 Core Vision
MoodMirror isn't just a journal; it's a **Neural Terminal**. We've replaced generic UI patterns with an immersive, data-driven experience that treats every mood entry as a "Synaptic Event."

---

## 🚀 Key Features

### 👤 User Node (The Experience)
- **Neural Synchronization:** Immersive onboarding with custom design palettes (Lavender, Slate, Obsidian) and emotional tokens.
- **Dynamic Journaling:** Full-screen, distraction-free journaling interface with "Pulse Mapping" for emotional intensity.
- **Sync History:** Premium calendar views visualizing emotional stability over time.
- **Zen Garden:** A specialized wellness module for meditative recalibration.
- **SOS Protocol:** Urgent emotional support triggers for high-stress events.

### 🛡️ Admin Node (The Command Center)
- **Operations Center:** Full-spectrum monitoring of system activity.
- **Entity Registry:** Management of all synchronized user nodes.
- **Synaptic Feedback:** Real-time user intelligence stream with sentiment ratings.
- **System Activity:** Live metrics tracking journal throughput and system health.

---

## 💻 Tech Stack
- **Frontend:** React.js, TailwindCSS, Lucide Icons, Axios *(Developed by Teammate)*
- **Backend:** Laravel (PHP 8.2+), RESTful API Architecture *(Developed by Me)*
- **Database:** MongoDB
- **Security:** Bearer-Token Authentication, OTP Email Verification
